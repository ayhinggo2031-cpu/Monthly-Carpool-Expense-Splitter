# Monthly-Carpool-Expense-Splitter

##Description
Phyton program that calculates monthly carpool expenses, divides costs among passengers (excluding the driver), and shows percentage breakdowns for transparency.

##Features
- Input validation for numeric values
- Percentage breakdown of expenses
- Loop for multiple months
- Functions for modular design
- Clear input-process-output structure

##How to Run
1. Clone the repository
2. Open 'main.py' in any Phyton IDE
3. Run the program and follow the prompts
